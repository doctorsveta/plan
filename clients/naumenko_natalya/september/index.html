    Простой завтрак с железом. Можно с красной фасолью из банки или орехами.

    Залить кипятком из термоса. Добавьте яйцо вкрутую из контейнера.
  • Черника и орехи (перекусы)
    Горсть черники для глаз, орехи для железа. Порционно в пакетиках.
